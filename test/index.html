Eventos — API Manual

Page View manual

Dispara un pageview sin navegar. Útil en SPAs donde la URL cambia sin recargar.

umami.track()

Evento simple

Registra un evento con nombre sin propiedades adicionales.

umami.track('boton-cta')

Evento con propiedades

Registra un evento con datos personalizados (plan, valor, etc.).

umami.track('upgrade', { plan: 'pro', precio: 29.99, moneda: 'USD' })

Evento de revenue

Registra una conversión con valor económico.

umami.track('compra', { revenue: 149.00, producto: 'plan-anual', descuento: 'PROMO20' })

Pageview con URL personalizada

Override de URL y título para tracking de pasos en un flujo.

umami.track(props => ({ ...props, url: '/checkout/paso-2', title: 'Checkout — Datos de pago' }))

Identify — identificar usuario

Asocia la sesión anónima con un usuario conocido (post-login).

umami.identify('user-42', { nombre: 'Ana García', plan: 'pro', empresa: 'ACME S.A.' })
